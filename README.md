## 🌙 Overview

**Oneiros** is an interactive AI-powered application that analyzes and interprets your dreams. Describe your dream through text or voice recording, and let the AI reveal hidden symbols, deep emotions, and visually illustrate your dreamworld.

---

## ✨ Features

- 🖊️ **Text input** — Describe your dream directly in the interface
- 🎙️ **Voice recording** — Speak your dream, the AI transcribes it automatically (Groq Whisper)
- 📖 **Structured summary** — Extraction of key characters, places, emotions and symbols
- 🔮 **Symbolic interpretation** — Narrative and symbolic analysis generated by LLaMA 3
- 🎨 **AI illustration** — Generation of a dreamlike image illustrating the scene (Stable Diffusion XL)
- 📓 **Dream journal** — Browsable history of all your past dreams

## ⚙️ Installation

**1. Clone the repository**
```bash
git clone https://github.com/your-username/oneiros.git
cd oneiros
```

**2. Create and activate virtual environment**
```bash
python -m venv .dreams_env
# Windows
.dreams_env\Scripts\activate
# Mac/Linux
source .dreams_env/bin/activate
```

**3. Install dependencies**
```bash
pip install -r requirements.txt
```

**4. Set up API keys**

Create a `.env` file at the root :
```
GROQ_API_KEY=gsk_xxxxxx
STABILITY_API_KEY=sk-xxxxxx
```

**5. Create the dream journal**

Create the `dreams_data/` folder and a `journal.json` file :
```json
{
    "dreams": []
}
```

**6. Run the application**
```bash
streamlit run app.py
```
